# PRD (Product Requirements Document): 충북대학교 경영정보학과 홍보 사이트

## 1. 프로젝트 개요 (Project Overview)
- **프로젝트 명**: 충북대학교 경영정보학과(MIS) 학과 소개 사이트
- **목적**: 학과의 비전, 역사, 교수진 정보를 20대 대학생 및 예비 신입생들에게 감각적이고 효과적으로 전달하는 단일 페이지 웹사이트 구축.
- **주요 메시지**: "미래를 선도하는 디지털 혁신 리더", "따뜻한 열정, 끝없는 혁신"

## 2. 타겟 오디언스 (Target Audience)
- **주 타겟**: 20대 대학생 및 예비 신입생 (MZ세대)
- **요구사항**: 모바일 최적화, 세련되고 감각적인 UI/UX, 정보의 직관적인 배치, 빠른 로딩 속도.

## 3. 웹사이트 구조 (Sitemap - Single Page)
1. **히어로 섹션 (Hero Section)**: 따뜻하고 신뢰감 있는 이미지 + 슬로건.
2. **학과 소개 (About US)**: 학과장 인사말 및 학과 핵심 가치.
3. **연혁 (History)**: 시각 자료를 활용한 시간순 연표.
4. **교육 목표 (Goals)**: 4가지 핵심 인재상 및 역량 소개.
5. **교수진 소개 (Faculty)**: 학과장 및 주요 교수진 프로필.
6. **문의하기 (Contact)**: 연락처, 팩스, 사무실 주소 및 학과 공식 홈페이지 링크.

## 4. 디자인 요구사항 (Design Specs)
- **색상 팔레트**: 따뜻한 베이지, 부드러운 블루, 그린 계열 (신뢰감과 따뜻함의 조화).
- **타이포그래피**: 
  - 읽기 쉬운 큰 글씨 (Clean Sans-serif 계열 추천).
  - 가독성을 중시한 줄 간격 및 자간 조절.
- **이미지**: 
  - Placeholder 사용 (alt 텍스트 필수).
  - 실제 사진 적용 시 따뜻한 톤의 보정 필요.
- **UX/UI 특성**:
  - **입체적 디자인**: 카드 요소 등에 3D 효과 및 다중 그림자 적용 (1.0 버전의 피드백 반영).
  - **반응형**: 모바일, 태블릿, 데스크탑 전 기기 최적화.
  - **애니메이션**: 스크롤 애니메이션(Fade-in, Lift-up), 마우스 오버 인터랙션 (Smooth transition).

## 5. 기술 요구사항 (Technical Specs)
- **언어**: HTML5, CSS3, Vanilla JavaScript.
- **특징**: 외부 라이브러리 최소화 (필요 시 CDN 사용), 단일 HTML 파일 구조 선호.
- **호스팅/배포**: GitHub Pages (main 브랜치).

## 6. 상세 콘텐츠 (Content Details)
- **교육 목표**: 커뮤니케이션 능력, 시스템 사고 능력, 창의적/윤리적 의사결정, 실무기반 전문능력.
- **수치적 성과**: 취업률 국내 1위(100%), 학과 선호도 국내 1위 목표.
- **역사적 포인트**: 1987년 국립대 최초 설립, 누리사업 선정(40억 지원), 경영교육 인증 최초 취득.

## 7. 단계별 개발 계획 (Development Roadmap)
1. **[Phase 1] 기획 및 분석**: PRD 확정 및 디자인 시스템 구축.
2. **[Phase 2] 구조 설계**: HTML 시맨틱 마크업 수행.
3. **[Phase 3] 스타일링**: CSS 변수 설정 및 반응형 레이아웃 구현 (3D 효과 포함).
4. **[Phase 4] 기능 구현**: JS 활용 스크롤 효과 및 모바일 메뉴 구현.
5. **[Phase 5] 검증 및 배포**: 크로스 브라우징 테스트 및 GitHub 푸시.
